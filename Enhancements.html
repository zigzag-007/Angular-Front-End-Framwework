Implemented Enhancements

We have implemented various HTML and CSS tags, including:

  • Strong: Bolds text within the tag.
  • Position: Used in most of the codes to fix positioning.
  • Keyframes: To display animations.
  • href: Links hyperlinks within the HTML.
  • Class selectors: Used to apply styles to elements of the same class. Examples include .page-1-content, .about-page-content, .quiz-page-content, .GOBack.
  • Heading tags: Used to define HTML headings. Examples include h1, h2.
  • List tags: Used for ordered lists (ol) and list items (li).
  • Notify-button: A class selector used to style the element with class “notify-button”.
  • Other class selectors: Used to style elements with the respective classes. Examples include .logo, .navbar, .icons, .fa-bars, .first-ul, .second-ul.
